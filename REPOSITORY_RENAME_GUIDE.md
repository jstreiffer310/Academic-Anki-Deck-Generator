## GitHub Rename Steps

### Option 1: GitHub Web Interface
1. Go to repository **Settings**
2. Scroll down to **Repository name**
3. Change from: `PSYC2240-Anki-Deck-Generator`
4. Change to: `Academic-Anki-Deck-Generator`
5. Click **Rename**

